# website-vibe-code-practice-cursor

Practice project for a redesigned portfolio site (static HTML/CSS/JS).

For this project, it is utilizing Cursor to redesign as a functioning prototype my original portfolio website. For this project, it is to experiment with vibecoding and AI generation skills but also to see what is possible in using AI as a tool to help with software development effiency to focus on developing solutions.

Some aspects in the prototype that Cursor was able to develop includes, loading animations, to create a functioning desktop and mobile version of the prototype website, a modernized user interface design, written, audio, and image placement, and creating the general framework of the website utilizing content from the original website that I created.

Due to only having a free Cursor plan, for this experiement, any changes made after April 3rd, 2026 at 5:10 PM are small changes only for quality of life functionality. An example of this is with loading content as when the pages were generated, Cursor only created it to the extent that the div folders for the section reveal tags that are used on the original website for the fade in/fade out content animations would load for the first website item. To resolve this since it was beyond the allowed context in the free plan, I have gone in and edited the code to make the animation work for as many projects as possible but it resolved to remove several section tags on older projects lower in the page to ensure that the content that Cursor generated would show up properly on the page. Another issue that occurred from Cursor was the buttons on the Projects Landing page initially overlapping which required changing the transform: scale() values.

All written content's origins that it was based on, image, audio, and video content on the page itself along with GitHub repository commits edits after April 3rd, 2026 at 5:10 PM are fully original with the animations, page structure, general code frameworks being from Cursor for this experiment. 

Additionally for this experiment, the image pop-up and close feature with image captions that appear when clicking on an image that I have on the original website does not have full functionality to keep authenticity to see what Cursor could generate in vibe coding. However, the images can still be viewed when scrolling and if an image is clicked on the Cursor prototype, please refresh the page to return to the rest of the page's content.
